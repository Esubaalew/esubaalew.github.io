## Local Development

```bash
pip install -r requirements.txt
mkdocs serve
```

Open [http://127.0.0.1:8000](http://127.0.0.1:8000)

## Build

```bash
mkdocs build
```

## Deploy

The site automatically deploys to GitHub Pages via GitHub Actions on every push to `main`.
